 Proyecto de Tienda Virtual (Java + JavaFX)

Este proyecto es una *aplicación de tienda virtual* desarrollada en *Java* utilizando *JavaFX* para la interfaz gráfica.  
Permite gestionar usuarios, productos, ventas y mostrar distintas vistas mediante archivos FXML y controladores organizados.


 Características principales

Gestión de usuarios
- Registro de usuarios
- Inicio de sesión
- Validación de credenciales  
- Los datos se guardan en usuarios.txt

Gestión de productos
- Carga de productos desde archivo
- Visualización de productos
- Funciones de búsqueda y filtrado
- Los datos se guardan en productos.txt

Interfaz gráfica (JavaFX)
- Múltiples ventanas creadas con *Scene Builder*
- Controladores independientes
- Navegación entre vistas
- Incluye imágenes y recursos necesarios
